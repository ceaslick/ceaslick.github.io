# EECS 440 Projects
Project 1: Computer Lab Simulation (VR)
- Experience a recreation of an old University of Michigan computer lab. Complete with interactive white board, fellow students, and a mysterious energy drink. 

Project 2: Location-based Game Inspired by Pokemon GO (AR) 
- Explore the city of Ann Arbor with an avatar that matches your real-life location. Plant trees and fend off squirrels as you seek to unlock all seeds by visiting local landmarks. 

Project 3: Michigan AR Classroom Companion (AR)
- Review course material by interacting with an AR pet through pop-up quizzes and scavenger hunt–style gameplay.
